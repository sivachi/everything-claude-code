# ハローワーク採用〜入社チェックリスト

ハローワーク経由で人材を採用する際の、募集から入社後手続きまでの全ステップをチェックリストとして出力するスキル。

## When to Activate

- ハローワークに求人を出すとき
- 新しい社員の採用が決まったとき
- 入社手続きの漏れがないか確認したいとき
- 社労士との連携タスクを整理したいとき
- キャリアアップ助成金を利用した採用を行うとき

## How It Works

ユーザーから採用イベントのトリガーを受けたら、以下の7ステップのチェックリストをMarkdown形式で出力する。
候補者名・入社日・助成金利用の有無など、ユーザーから情報が提供された場合はテンプレートに埋め込む。
情報がない場合はプレースホルダー（`【候補者名】`等）で出力する。

## テンプレート

### STEP 1｜募集・応募受付

- [ ] ハローワークに求人票を提出・掲載
- [ ] 応募者から書類受付（履歴書・職務経歴書・ハローワーク紹介状）
- [ ] 書類選考

### STEP 2｜面談

- [ ] 面談日程調整（メールにて候補日提示）
- [ ] Google Meet カレンダー作成・URLを候補者に送付
- [ ] 面談実施（オンライン or 対面）
- [ ] 社内で採否の検討

### STEP 3｜内定〜承諾

- [ ] 内定通知メール送付
- [ ] 内定承諾の受領確認
- [ ] ハローワーク（採用側）で選考結果を登録（採否・職種・雇用予定日・賃金・採用理由を入力）
- [ ] ハローワーク（応募者側）へ選考結果を通知（マイページ「求職者に選考結果を連絡」ボタンから）

### STEP 4｜契約書類の準備・締結

> ⚠️ キャリアアップ助成金（40万円）取得の場合は、有期雇用6ヶ月→正社員転換スキームで対応すること。
> 雇用契約書は社労士が作成。

- [ ] フレックスタイム制に関する労使協定書の確認（既存の締結済みが有効か社労士に確認）
- [ ] **【社労士より】** 有期雇用版 雇用契約書の受領
- [ ] 雇用契約書を候補者にメールで送付・確認依頼
- [ ] 候補者からOK返信の受領
- [ ] 承諾確認後、クラウドサインで電子署名締結（雇用契約書のみ）
- [ ] **【社労士連携】** 締結済み雇用契約書を社労士に送付（社会保険加入手続きのため）

### STEP 5｜入社前の情報収集

- [ ] マイナンバーカード（または通知カード）の画像受領（表裏）→ 社労士に送付
- [ ] 雇用保険被保険者証のコピー受領（紛失の場合は再発行手続きを依頼）
- [ ] 給与振込先の口座情報受領（銀行名・支店・口座番号）
- [ ] **【社労士連携】** 雇用保険被保険者証を社労士に送付
- [ ] PC環境確認（光回線 or Wi-Fi、希望OS：Windows / Mac）
- [ ] 入社日・初出勤の段取り案内メール送付

### STEP 6｜入社日当日

- [ ] ハローワーク提出用書類に会社印を押印して手渡し
- [ ] 会社概要・業務内容・社内ルールの説明
- [ ] PC セットアップ・貸与（希望OS：Windows / Mac）
- [ ] その他貸与品の準備・手渡し（社員証・名刺等）
- [ ] 社内ツールのアカウント発行・案内（Slack、Google アカウント等）

### STEP 7｜入社後の手続き

- [ ] **【社労士対応】** 社会保険（健康保険・厚生年金）加入申請 ← 社労士が実施
- [ ] **【社労士対応】** 雇用保険 加入申請 ← 社労士が実施（労働保険成立後、数週間かかる）
- [ ] 健康保険証（資格確認書）の到着確認（申請から約2週間）
- [ ] 雇用保険事業所番号の受領確認（社労士より連絡あり）
- [ ] 源泉徴収・給与設定を給与ソフトに登録
- [ ] 給与振込先を給与ソフトに登録
- [ ] マイナンバーを労働者名簿に記録・保管
- [ ] 名刺作成・手配
- [ ] **【社労士連携・毎月】** 給与支払い月末前に勤怠情報・インセンティブ有無を社労士に送付（給与計算依頼）
- [ ] **【社労士連携】** ネットde顧問のユーザー登録書を受領 → 入社者に案内（給与明細閲覧用）
- [ ] 初月給与の確認・調整（試用期間条件の反映）
- [ ] **【キャリアアップ助成金】** 有期雇用6ヶ月経過後、正社員転換手続き（3%昇給必要）← 社労士が申請代行（助成金40万円）

## Examples

### 例1: 基本的な使い方

```
ユーザー: 新しい人を採用するので入社までのチェックリストを出して
→ 上記テンプレートをそのまま出力
```

### 例2: 候補者情報付き

```
ユーザー: 田中太郎さんを4月1日入社で採用する。キャリアアップ助成金を使う
→ テンプレートに候補者名・入社日を埋め込み、助成金の注意書きを強調して出力
```

### 例3: 特定ステップの確認

```
ユーザー: 入社後の社労士連携タスクだけ教えて
→ STEP 7の社労士関連タスクのみ抽出して出力
```
