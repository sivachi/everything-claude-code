# freee 標準経費科目一覧

## よく使われる経費科目（Discord選択肢）

| コード | 科目名 | 説明 |
|-------|--------|------|
| SUPPLIES | 消耗品費 | 10万円未満の消耗品、備品など |
| OFFICE_SUPPLIES | 事務用品費 | 文房具、事務用消耗品など |
| TRAVEL | 旅費交通費 | 電車賃、バス代、タクシー代、出張費など |
| MEETING | 会議費 | 会議・打ち合わせの飲食代など |
| ENTERTAINMENT | 交際費 | 接待交際費、お中元・お歳暮など |
| BOOKS | 新聞図書費 | 新聞、書籍、雑誌、資料など |
| COMMUNICATION | 通信費 | 電話代、インターネット代、郵送費など |
| UTILITIES | 水道光熱費 | 電気、ガス、水道代など |
| COMMISSION | 支払手数料 | 振込手数料、各種手数料など |
| VEHICLE | 車両費 | ガソリン代、駐車場代、車両維持費など |
| RENT | 地代家賃 | 事務所・店舗の家賃、地代など |
| ADVERTISING | 広告宣伝費 | 広告、宣伝活動費用 |
| OUTSOURCING | 外注費 | 外部委託費用 |
| REPAIR | 修繕費 | 設備・機器の修理費用 |
| MISC | 雑費 | その他の少額経費 |

## 全経費科目一覧

### 人件費系
- TRAINING: 研修費
- WELFARE: 福利厚生費
- RECRUITMENT: 採用教育費

### 営業・販売系
- OUTSOURCING: 外注費
- SHIPPING: 荷造運賃
- ADVERTISING: 広告宣伝費
- ENTERTAINMENT: 交際費
- MEETING: 会議費
- TRAVEL: 旅費交通費
- COMMUNICATION: 通信費
- SALES_COMMISSION: 販売手数料
- PROMOTION: 販売促進費

### 事務・管理系
- SUPPLIES: 消耗品費
- OFFICE_SUPPLIES: 事務用品費
- REPAIR: 修繕費
- UTILITIES: 水道光熱費
- BOOKS: 新聞図書費
- MEMBERSHIP: 諸会費
- COMMISSION: 支払手数料
- VEHICLE: 車両費

### 不動産・賃貸系
- RENT: 地代家賃
- RENTAL: 賃借料
- LEASE: リース料

### その他
- INSURANCE: 保険料
- TAX: 租税公課
- PROFESSIONAL: 支払報酬料
- DONATION: 寄付金
- FUEL: 燃料費
- R_AND_D: 研究開発費
- MISC: 雑費
- MANAGEMENT: 管理諸費

## 自動分類ルール

主要なキーワードによる自動分類：

### コンビニ・スーパー → 消耗品費
- セブンイレブン、ファミリーマート、ローソン、ミニストップ

### 交通機関 → 旅費交通費
- JR、電車、タクシー、バス、新幹線、航空、ホテル

### 飲食店 → 会議費/交際費
- レストラン、カフェ（会議費）
- 居酒屋（交際費）

### 書店 → 新聞図書費
- 書店、本屋、紀伊國屋、ジュンク堂、Amazon（書籍）

### ガソリンスタンド → 車両費
- エネオス、出光、駐車場

### 通信会社 → 通信費
- ドコモ、au、ソフトバンク、郵便

### 公共料金 → 水道光熱費
- 電気、ガス、水道
