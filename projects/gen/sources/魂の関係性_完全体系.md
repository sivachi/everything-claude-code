# 竹尾メソッドにおける「魂の関係性」完全体系

ソースから抽出された情報を基に、**キラキラ**と**ソウルメイト7分類**の全体像、出会いの条件、関係性の進展、そして分界点を整理する。

---

## 1. ソウルメイトの7段階ピラミッド（多い順→少ない順）

ソースの中で最も体系的に語られているのは `20230414錦糸町謎の会.m4a` である。

| # | 名称 | 世界での人数 | 定義（ソースからの直接引用ベース） |
|---|---|---|---|
| 7 | **ディバインコンプリメント** | 不特定多数 | 「鏡のような存在」。自分を映し出す相手。最も出会いやすい |
| 6 | **ディバインエクスプレッション** | 数千人 | 「あなたに学びを与え、互いに成長し合う存在」。ソウルメイトと混同しやすい |
| 5 | **ライトパーソン** | （ソース内で詳述なし） | 正しい道へ導く存在と推定 |
| 4 | **ツインメイト** | 世界で**144人** | 「この世に同じ役割を持って生まれた存在」。ビジネスパートナーや結婚相手になりやすい |
| 3 | **ツインソウル** | 世界で**12人** | 「極めて近い周波数を持った存在」。初対面で「なんか近いな」と感じる |
| 2 | **ツインフレーム** | 世界で**7人** | 「前世も今世も全部共に歩んできた存在」。何度も同じような人生を歩んできた |
| 1 | **ツインレイ** | 世界で**1人** | 「魂の片割れ。1つの魂を分け合った存在」。最も出会いにくい |

> **竹尾さんの統計**: 80億人中、ツインレイに会えるのは約**100人**ほど。「1度会ったら2度会うことはなかなか難しい」

> **上位3段階（7〜6）までで数千人**。4のツインメイト以降、一気に希少になる。

## 2. 「キラキラ」の概念（ソウルメイト体系とは別軸）

**キラキラとツインレイは明確に「違う」と断言されている。**

> 「ツインレイとキラキラは違うんですね。」「うん。」（20221204四谷タケチング）

### キラキラの定義

| 要素 | 内容（ソースからの抽出） |
|---|---|
| **本質** | 「**ギフト（贈り物）**」「**ご褒美**」。頑張った人が得られる権利 |
| **人数** | 1人の人生で**最大3人**まで |
| **変更可否** | 「宿命のところで変えられない」＝**変更不可**。予定外の人がキラキラになることはない |
| **パーセンテージ** | **80%** / **100%** などの段階がある。100%が最高 |
| **使命との関係** | 「使命の道を外れて生きていたらキラキラに会えなかった。使命をやると決めた途端に会えた」 |
| **昇格の可能性** | 「2人とも課題の塊同士だから、昇格したら多分キラキラになる」＝条件付きで昇格あり。ただし100%への昇格は全員ができるわけではない |

### 190%の法則（キラキラ最重要法則）

> 「お互いに100っていう数字があって、向こうが10しか成長できてないってなると、自分は**190成長できてないと割に合わない**。」（20221204四谷タケチング）

つまり:
- キラキラの関係は **合計200（100+100）** が基準
- 相手の成長が10なら → 自分が**190**まで成長して補う必要がある
- **相手の不足分を自分の成長で埋める**という圧倒的な当事者意識

### キラキラに出会うための条件

1. **使命の道を歩いていること**（使命から外れていると会えない）
2. **十分な自己成長を遂げていること**（190%の法則）
3. **キラキラに会うこと自体を目的にしないこと**

> 「キラキラに会いたいからそっちに向くっていう邪な…会うことに執着をして、会うことが中心になって、使命を達成していこうっていう方に向けるエネルギーが失われちゃう」

4. **知らずに出会う方がいい**

> 「知らずに出会うっていう方が本当ねギフト感が増えるの。食べてみてって言われて食べた時の方が感動するんです」

## 3. ツインレイに出会うための絶対条件

ツインレイはソウルメイト体系の最上位であり、最も過酷な条件が課される。

### 出会いの条件

> 「逃げたり避けたりごまかしたり騙したり抜け道ってあると思うんですけど、**全て克服してこないと会えない人**でもありますね」

### 出会いのパラドックス（嫌悪感からのスタート）

> 「ツインレイだけは、会っても**ポジティブなイメージがない**。大体喧嘩します」
>
> 「自分の1番弱いところが相手の中に見えてくる。**嫌悪感**を感じます」
>
> 「嫌悪感から始まるんですよ。お互いに努力ができてれば『一緒に頑張り合いましょう』ってなるんですけど、できてない場合はもう嫌悪感でしかない。だから**せっかくツインレイで会えたのにほとんどの人はそこで嫌になって離れてしまう**」

> 「**99%は会っても切れちゃう**」（20221204四谷タケチング）

### 嫌悪感の奥にある愛

> 「無関心の対義語が愛であるように、感情があるってことは愛があるっていうこと。情よりもそのもう少し奥に存在する**愛情に冷静に降りていった時に**、もしかしたらツインレイかもしれない」

## 4. キラキラ vs ソウルメイト vs ツインレイ — 分界点の整理

| 判断軸 | キラキラ | ソウルメイト | ツインレイ |
|---|---|---|---|
| **体系** | **独立した別概念** | 7段階ピラミッドの総称 | ソウルメイト体系の最上位 |
| **人数** | 最大3人 | 数千人（下位）〜1人（上位） | 世界で1人 |
| **決定要因** | 宿命（変更不可） | 前世の約束・魂の構造 | 魂の片割れ |
| **出会いの感覚** | ギフト感・感動 | 共感・安心・似た課題 | **嫌悪感・激しい感情** |
| **出会いの条件** | 使命を歩む＋十分な成長 | 比較的会いやすい | 逃げ/ごまかしの完全克服 |
| **関係の性質** | ご褒美・成長促進 | 共に歩む・サポート | 魂の研鑽・最大の試練 |

| **パーセンテージ** | 80%〜100%の段階あり | なし | なし |

## 5. 出会いのタイミングとベストな関係の進め方

ソースを横断すると、以下のような**時系列の流れ**が浮かび上がる。

### Phase 1: 土台づくり（〜使命を見つけるまで）
- ディバインコンプリメント（鏡のような存在）と多く出会う
- ソウルメイト（前世の約束の存在）とも自然に出会う
- この段階では「課題が一緒・方向性が似てる」相手と共感し合う

### Phase 2: 使命への覚悟（使命を自覚し歩み始める）
- **使命の道を歩き始めると、キラキラに会える可能性が開く**
- 「使命の道に覚悟するスタートラインの時」にキラキラに会えた体験が語られている
- 現在のパートナーが「サルタ彦（導き手）」としてキラキラの近くに連れていってくれることもある

### Phase 3: 自己成長の深化（逃げ・ごまかしの克服）
- 自分の弱さ・逃げ・ごまかしを一つずつ克服していく
- 「道路作り」＝自分の使命を自分の足で歩ける基盤を作る
- この過程でツインメイト・ツインソウル・ツインフレームと出会う可能性あり

### Phase 4: キラキラとの出会い（ご褒美）
- 十分に成長した人への「権利」として訪れる
- 話者自身は「100%のキラキラに50代で会うために頑張っている」「下手したら70-80で会うかも」と語る
- **焦らない。結果論で出会っていくのがベスト**

### Phase 5: ツインレイとの出会い（最大の試練かつ最大のギフト）
- 人生のあらゆる逃げ・ごまかしを克服した後に訪れる可能性
- 嫌悪感の中に愛を見出せるかが最大の分岐点
- 「1度会ったら2度目はない」ため、その瞬間を大切にする

## 6. 「この人はキラキラなのかソウルメイトなのか」の判断基準

ソースから導出される**実践的な分界点**:

| チェックポイント | キラキラの場合 | ソウルメイトの場合 | ツインレイの場合 |
|---|---|---|---|
| 出会った時の感覚 | 「自分の意思とは関係なくくっつけられた感じ」 | 「共感・安心・似た方向性」 | 「嫌悪感マックス」 |
| 使命との連動 | 使命を歩み始めたタイミングで出会う | 使命と直接関係なく出会える | 逃げを全克服した後に出会う |
| 成長への影響 | 成長を加速させるが「癒されすぎると止まる」 | 共に課題に取り組める | 自分の最大の弱点と向き合わされる |
| 居心地 | ギフト感・感動 | 居心地が良い | 不快→深い愛への転換 |
| 人数 | 「あと何人」とカウントできる（最大3人） | 多数存在する | 1人しかいない |

## 7. なぜこれらの関係性が存在するのか（根本原理）

> 「どうしてディバインコンプリメントからツインレイまでいろんな関係性の人たちがいるかって言うと、**1人では幸せになれないから**。周波数が近かったり課題が同じだったりっていう人たちと共に歩みなさいっていうことを人間は求められてる」

> 「出会ったのがスタートラインで、そのスタートラインすら立てない人がほとんど。ソウルメイトとか結婚できたとか子供生まれたとか**全部スタートラインでしかない**。そっからが本番」
