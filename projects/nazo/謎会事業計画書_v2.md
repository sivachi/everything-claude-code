# 謎会 事業計画書

**作成日：** 2026年3月23日（改訂）
**事業主体：** 囲炉裏（既存事業の一部門としてスタート、将来法人化予定）

---

## 1. エグゼクティブサマリー

「謎会」は、スピリチュアル・未知の領域をテーマに、参加者が疑問を持ち寄り対話を通じて探求する**知的好奇心コミュニティ事業**である。

2018年の第0回開催から約8年、延べ30回以上のセッションを重ね、参加者の自己理解・成長を支援してきた。本計画では、これまで蓄積したセッションコンテンツと独自の知見を資産として活用し、**サブスクリプション型コンテンツ配信**を軸に、イベント参加費・記事販売を組み合わせた収益モデルで事業化する。

スモールスタートから段階的に拡大し、将来的には動画制作・映画・漫画・書籍出版・体験型イベントへと展開する。

## 2. 事業概要

### 2.1 ミッション

> **「謎は謎のままにしない」**
> 正しい・間違いの二項対立ではなく、多角的な視点で物事の本質に迫り、参加者が自分自身の感覚を取り戻す場を提供する。

### 2.2 事業コンセプト

- 世の中に存在する「謎」をテーマにした参加型ディスカッション・Q&Aイベント
- スピリチュアル・歴史・科学・社会を横断する**オールジャンルの謎**を扱う
- 一方的な講義ではなく、参加者が質問を持ち寄り、対話の中で気づきを得る双方向型
- 「謎を解決する」だけでなく**「謎を生み出す・発見する」**場（2024年以降のコンセプト進化）
- **「本気でふざける」** — ポップに、軽く、でも本質には深く迫るスタンス

> 「謎っていうスパイスは何かにかけてこそ生きる」 — 2023.8.16運営MTG
> 「肉体を持って生まれてきた限りは自分は何者なのかとかどうして生まれてきたのかとか、毎日の中死ぬまで謎との向き合い」 — 2023.4.9コンサル

### 2.3 扱うテーマ体系（セッション実績に基づく）

過去8年・30回以上のセッションをAI分析した結果、以下の9つの主要テーマに分類される知見が蓄積されている。

| # | テーマ | 割合 | セッションでの具体例 |
|---|--------|------|---------------------|
| 1 | 存在（魂・霊・神） | 18% | 守護霊の構成と導きの仕組み、神社の神々と崇拝、自然霊 |
| 2 | 能力・感覚 | 30% | 直感の磨き方、予知夢の見分け方、透視能力、バリアの張り方 |
| 3 | 人生・成長 | 16% | カルマと学び、魂の成長レベル、生まれ変わりの選択肢 |
| 4 | 関係性 | 7% | ツインレイ、男女のエネルギー、家族関係と魂の絆 |
| 5 | 実践・セルフケア | 10% | 状態切り替えのトリガー、オーラの浄化、宇宙エネルギーの受け取り |
| 6 | 社会・文明 | 1% | フリーメーソンと相場操作、集団心理と社会構造 |
| 7 | 宇宙・世界構造 | 10% | 次元の階層、アトランティス・ムーの滅亡、魂の振り分け |
| 8 | 現象・体験 | 5% | デジャブの意味、夢のメッセージ解釈、お盆時期の霊の来訪 |
| 9 | 知識・情報 | 3% | 神社・お寺・お墓参りの違い、除霊と成仏の違い |

※分析途中のデータに基づく（502/4,167チャンク分類済み）。最終的な比率は変動する可能性あり。

## 3. 事業の強み・独自性

### 3.1 8年分のセッション資産 — データで裏付けられた知見体系

2018年〜2025年に蓄積された30回以上のセッション書き起こしデータを保有。AI分析パイプラインにより、以下の規模でコンテンツ資産の構造化が進行中。

| 資産 | 規模 | 状態 |
|------|------|------|
| セッション書き起こし（謎会イベント） | 30回・17ファイル | 構造化済み |
| 運営MTG記録 | 14ファイル | 構造化済み |
| 音声文字起こし（録音データ） | 5件 | 3件完了、2件進行中 |
| 神社メッセージ（神様からの言葉） | 24ファイル・306エントリ・28神様/神社 | 構造化済み |
| GENプロジェクト知見（姉妹活動） | 258ファイル | 取り込み済み |
| **チャンク総数（AI分析単位）** | **4,167チャンク** | **分類進行中** |

これらの資産は9カテゴリ・35サブカテゴリに体系化され、テーマ別コンテンツとして即座に再利用可能な状態にある。

### 3.2 独自のファシリテーション

主催ファシリテーター（玄）が20年以上の経験を持ち、参加者の質問に対して即座に多角的な視点から回答できる。一般的なスピリチュアル講座が「教える」形式であるのに対し、謎会は**「一緒に探求する」**形式を取る。

### 3.3 コミュニティの質

参加者同士の関係性が深く、新規参加者も自然に受け入れられる風土がある。集団心理を良い方向に活用し、個人の気づきをコミュニティ全体の成長につなげる。シンクU内では「最もライト」なポジションで、他チームとの連携ハブとしても機能する。

| 一般的なスピリチュアル講座 | 謎会 |
|--------------------------|------|
| 講師が一方的に教える | 参加者が疑問を持ち寄り対話する |
| 特定のジャンルに限定 | オールジャンルの謎を扱う |
| 正解を提示する | 正しいも間違いもない中立の立場 |
| 受講して終わり | コミュニティとして継続的に関わる |

### 3.4 神社メッセージ資産

24回の神社参拝で受け取った神様からのメッセージを体系的に保有。28柱の神様/神社からの306件のメッセージが整理されており、神社ツアーや書籍化の独自コンテンツとなる。

## 4. サービス内容

### 4.1 サブスクリプション（月額会員制）

**収益の柱。コンテンツ提供を中心としたオンラインコミュニティ。**

| 項目 | 内容 |
|------|------|
| 月額料金 | 1,000円〜2,000円（段階的プラン検討） |
| 提供コンテンツ | 過去セッションの音声・テキスト配信、テーマ別解説コンテンツ、会員限定コラム |
| コミュニティ | 会員専用チャット（質問・交流）、会員同士のディスカッション |
| 特典 | イベント参加費の割引、新コンテンツの先行公開 |

### 4.2 謎会イベント（定期開催）

| 項目 | 内容 |
|------|------|
| 開催頻度 | 月1回〜2回 |
| 形式 | 対面（日曜日 13:00〜、2〜3時間）+ オンライン配信 |
| 参加費 | 1,500円〜5,000円（テーマ・形式に応じて変動、会員割引あり） |
| 内容 | 前半：事前質問への回答セッション / 後半：テーマ別ディスカッション |
| 特別形式 | クイズ形式、食イベント、五感刺激の演出付き体験型 |

**テーマ設定の方針**（運営MTGでの方針）:
- **トレンド連動**: 時事ネタ・季節ネタへの即応（WBCの謎、花火の謎、自由研究等）
- **日常の謎**: コカ・コーラはなぜ黒いのか、添加物の謎
- **ディープ系**: 生と死、社会構造の裏側
- **テーマ別価格**: 高付加価値テーマは参加費を上げ、カフェ等の特別会場で開催

### 4.3 記事販売（note）

| 項目 | 内容 |
|------|------|
| プラットフォーム | note |
| 価格帯 | 300円〜1,000円/記事 |
| 内容 | セッション内容を編集・体系化した記事、テーマ別深掘り解説 |
| 更新頻度 | 週1〜2本 |

### 4.4 SNS・動画戦略

運営MTGで繰り返し重要性が確認されている領域。認知獲得→ファン獲得→イベント集客の導線を構築する。

| チャネル | 方針 |
|----------|------|
| 音声SNS（Voicy, stand.fm等） | 先行者優位を狙う。10分の短い番組、日替わりパーソナリティ |
| X / Instagram | テーマに関する短いコンテンツを定期投稿 |
| YouTube | セッションダイジェスト動画、テーマ別解説動画 |
| TikTok | 「本気でふざける」スタイルの短尺動画 |
| VTuber / アバター | 顔出し不要のキャラクター発信（猫キャラクター構想あり） |

### 4.5 将来展開（フェーズ2以降）

- **書籍出版**: セッション内容を書籍化（自費出版20万円〜）、イベント手売り + オンライン販売
- **漫画制作**: 謎会の内容をストーリー化した漫画コンテンツ
- **映画制作**: 「謎で始まって謎で続いて謎で終わる」ドキュメンタリー / 劇映画
- **体験型ツアー**: UFOスポット探訪、埋蔵金探し、パワースポット巡り、神社ツアー
- **アプリ開発**: 「徳積み」ゲームアプリ（人生ゲーム形式で徳を積むとポイント/ガチャが回せるシステム）
- **シンクU連携**: 他チームの専門性×謎で新しい価値を創出（ドリーム×謎、愛×謎、教育×謎）

## 7. マーケティング戦略

### 7.1 フェーズ1：認知獲得（0〜6ヶ月）

- **note記事の定期投稿**で検索流入を獲得 — 9カテゴリの知見体系から記事テーマを量産
- **音声SNS**（Voicy, stand.fm等）での情報発信 — 先行者優位を取りやすい
- **SNS発信**（X, Instagram）でテーマに関する短いコンテンツを投稿
- 既存参加者からの**紹介割引制度**で口コミ促進（紹介者の参加費をペイできる仕組み）
- **トレンド連動コンテンツ**: 時事ネタ・季節ネタに即応した発信

### 7.2 フェーズ2：コンテンツ拡充（6〜18ヶ月）

- 過去セッションのアーカイブを整理・編集してサブスクコンテンツ化（4,167チャンク分の知見を活用）
- YouTube動画制作を開始（セッションダイジェスト、テーマ解説）
- ポッドキャスト配信でリーチ拡大
- 書籍出版（初版は自費出版、イベント手売り + オンライン販売）
- **テーマ別コンテンツシリーズ化**: 「守護霊の仕組み」「夢とメッセージ」「魂の成長」等、セッションデータから体系化された知見を連載

### 7.3 フェーズ3：拡大・多角化（18ヶ月〜）

- 体験型ツアーの定期開催（神社ツアーは306件の神社メッセージ資産と連動）
- 漫画・映像コンテンツの制作
- 企業・団体とのタイアップイベント
- コミュニティ内で自然発生するイベントの手数料モデル
- **コールアンドレスポンス型集客**: 謎を生み出す発信 → 解決したい人が集まる導線の確立

## 8. 運営体制

### 8.1 初期メンバー

| 役割 | 担当 | 主な業務 |
|------|------|---------|
| 事業統括・企画 | 倉田（クラッチ） | 全体運営、コンテンツ制作、IT・AI活用 |
| ファシリテーター | 主催者（玄） | セッション進行、質疑応答、知見提供 |
| 運営サポート | 満美 | イベント企画・運営補助 |
| 運営サポート | ルミコ | コミュニティ管理・運営補助 |

### 8.2 外部リソース

- **AI活用**（セッション書き起こし分析、コンテンツ生成支援、FAQ自動生成）
  - ただし「心がこもっていない」「不特定多数向けの内容になりがち」というリスクあり。AI生成はあくまで下書き・素材として活用し、人の手で仕上げる方針
- デザイン・編集は外注（必要に応じて）
- IT/ゲーマー人材の探索（徳積みアプリ開発用、将来的に）

### 8.3 運営の基本スタンス

運営MTGで繰り返し確認されている原則:

- **楽しさ最優先**: 「運営も楽しくないと続かない。タスクをこなしてる感じにしない」
- **サークル的**: 会社でも部活でもなくサークルに近い雰囲気
- **テーマパーク的体験**: 学ぶより体験する。五感を使った演出も重要
- **NGなし、壁なし**: 何でもあり、ノージャッジ

## 12. コンテンツ資産の活用計画

8年分のセッション・運営MTG・神社メッセージをAI分析パイプラインで構造化。4,167チャンクを9カテゴリ・35サブカテゴリに分類し、テーマ別に検索・再利用可能な状態にした。

### 12.2 即座にコンテンツ化できるテーマ群

セッションデータの分析から、以下のテーマは既に十分な知見が蓄積されており、note記事・書籍の章立てとして即座に展開可能。

| テーマ | チャンク数 | コンテンツ化の方向性 |
|--------|-----------|---------------------|

## 13. 社会的貢献

### 13.1 自己認識の深化

参加者が自分自身の性質・感情・価値観を理解し、主体的に人生を選択できるよう支援する。自己理解の深化は、他者への理解・多様性の受容につながる。

### 13.2 コミュニケーションの質向上

「伝える」と「伝わる」の違いを体感的に学ぶ場を提供する。対話を通じて、固定観念に囚われない柔軟な思考と、他者に寄り添うコミュニケーション力を育む。

### 13.3 知的好奇心の喚起

「謎は謎のままにしない」という姿勢で、参加者の知的好奇心を刺激する。正解を追い求めるのではなく、問いを持ち続けること自体の価値を伝える。

### 13.4 コミュニティによる孤立防止

同じ関心を持つ人々がつながり、支え合うコミュニティの形成を通じて、現代社会における精神的孤立を防ぐ。
