## 1. 障害者手帳の種類と等級の全体像

### 1-1. 身体障害者手帳

| 等級 | 程度 | 障害者区分 |
|------|------|-----------|
| **1級** | 最重度 | **特別障害者** |
| **2級** | 重度 | **特別障害者** |
| 3級 | 中度 | 一般障害者 |
| 4級 | 中度 | 一般障害者 |
| 5級 | 軽度 | 一般障害者 |
| 6級 | 軽度 | 一般障害者 |
| 7級 | 最軽度 | 単独では手帳交付なし（2つ以上の7級で6級として交付） |

**障害の種類と該当等級:**

| 障害の種類 | 該当等級 |

| 音声・言語・そしゃく機能障害 | 3級、4級 |
| 肢体不自由（上肢） | 1級〜7級 |
| 肢体不自由（下肢） | 1級〜7級 |
| 肢体不自由（体幹） | 1級〜3級、5級 |
| 肢体不自由（脳原性運動機能障害） | 1級〜7級 |
| 心臓機能障害 | 1級、3級、4級 |
| じん臓機能障害 | 1級、3級、4級 |
| 呼吸器機能障害 | 1級、3級、4級 |
| ぼうこう又は直腸機能障害 | 1級、3級、4級 |
| 小腸機能障害 | 1級、3級、4級 |
| ヒト免疫不全ウイルスによる免疫機能障害 | 1級〜4級 |
| 肝臓機能障害 | 1級〜4級 |

### 1-2. 精神障害者保健福祉手帳

| 等級 | 程度 | 障害者区分 |
|------|------|-----------|
| **1級** | 重度（日常生活の用を弁ずることを不能ならしめる程度） | **特別障害者** |
| 2級 | 中度（日常生活が著しい制限を受ける程度） | 一般障害者 |
| 3級 | 軽度（日常生活・社会生活が制限を受ける程度） | 一般障害者 |

### 1-3. 療育手帳（知的障害）

| 等級 | 程度 | 障害者区分 | 判定基準 |
|------|------|-----------|---------|
| **A1（最重度）** | 最重度 | **特別障害者** | IQ おおむね20以下 |
| **A2（重度）** | 重度 | **特別障害者** | IQ おおむね21〜35 |
| B1（中度） | 中度 | 一般障害者 | IQ おおむね36〜50 |
| B2（軽度） | 軽度 | 一般障害者 | IQ おおむね51〜75 |

※ 自治体により表記が異なる場合あり（東京都は「愛の手帳」、1度〜4度で表記）

## 2. 特別障害者 / 一般障害者の分類基準

### 特別障害者に該当する方

| 手帳の種類 | 該当条件 |
|-----------|---------|
| 身体障害者手帳 | **1級または2級** |
| 精神障害者保健福祉手帳 | **1級** |

| その他 | 精神上の障害により事理を弁識する能力を欠く常況にある方 |
| その他 | 6ヶ月以上寝たきりで複雑な介護を必要とする方 |

### 一般障害者に該当する方

| 手帳の種類 | 該当条件 |
|-----------|---------|
| 身体障害者手帳 | **3級〜6級** |
| 精神障害者保健福祉手帳 | **2級または3級** |
| 療育手帳 | **B判定**（B1・B2） |

## 3. 身体障害者手帳 第1種・第2種の分類

第1種・第2種は主に **旅客運賃割引** のための区分です。

### 分類基準

| 障害の種類 | 第1種 | 第2種 |
|-----------|-------|-------|
| 視覚障害 | 1級〜3級、4級の一部 | 4級の一部、5級〜6級 |
| 聴覚障害 | 2級〜3級 | 4級、6級 |
| 平衡機能障害 | − | 3級、5級 |
| 音声・言語・そしゃく機能障害 | − | 3級、4級 |
| 肢体不自由（上肢） | 1級〜2級 | 3級〜6級 |
| 肢体不自由（下肢） | 1級〜2級（一部3級含む） | 3級〜6級 |
| 肢体不自由（体幹） | 1級〜3級 | 5級 |
| 心臓・じん臓・呼吸器・ぼうこう直腸・小腸・肝臓・免疫機能障害 | 1級、3級 | 4級 |

### 第1種・第2種による違い

| 項目 | 第1種 | 第2種 |
|------|-------|-------|
| JR運賃割引（本人） | 50%割引 | 50%割引（片道101km以上） |
| JR運賃割引（介護者） | **50%割引** | 対象外 |
| 私鉄・バス割引（本人） | 50%割引 | 50%割引 |
| 私鉄・バス割引（介護者） | **50%割引** | 対象外 |
| 高速道路ETC割引 | **50%割引** | **50%割引** |
| タクシー割引 | 10%割引 | 10%割引 |

## 4. 税制優遇一覧（等級別）

### 4-1. 所得税

| 控除項目 | 特別障害者 | 一般障害者 |
|---------|-----------|-----------|
| 障害者控除 | **40万円**（所得控除） | **27万円**（所得控除） |
| 同居特別障害者加算 | **75万円**（扶養控除に加算） | − |

**対象:**
- 特別障害者: 身体1〜2級 / 精神1級 / 療育A
- 一般障害者: 身体3〜6級 / 精神2〜3級 / 療育B

### 4-2. 住民税

| 控除項目 | 特別障害者 | 一般障害者 |
|---------|-----------|-----------|
| 障害者控除 | **30万円**（所得控除） | **26万円**（所得控除） |
| 同居特別障害者加算 | **53万円**（扶養控除に加算） | − |
| **非課税措置** | 前年合計所得 **135万円以下** で非課税 | 前年合計所得 **135万円以下** で非課税 |

### 4-3. 相続税

| 区分 | 控除額 |
|------|-------|
| 特別障害者 | 85歳までの年数 × **20万円** |
| 一般障害者 | 85歳までの年数 × **10万円** |

**計算例:**

| 年齢 | 特別障害者の控除額 | 一般障害者の控除額 |
|------|------------------|------------------|
| 30歳 | (85-30) × 20万 = **1,100万円** | (85-30) × 10万 = **550万円** |
| 40歳 | (85-40) × 20万 = **900万円** | (85-40) × 10万 = **450万円** |
| 50歳 | (85-50) × 20万 = **700万円** | (85-50) × 10万 = **350万円** |
| 60歳 | (85-60) × 20万 = **500万円** | (85-60) × 10万 = **250万円** |

### 4-4. 贈与税（特定贈与信託）

| 区分 | 非課税限度額 |
|------|------------|
| 特別障害者（身体1〜2級 / 精神1級 / 療育A） | **6,000万円** |
| 特定障害者（精神2〜3級 / 療育B等） | **3,000万円** |

**特定贈与信託の詳細:**

- **委託者**: 個人であれば誰でもOK（親族でなくても、**知人・他人でも可能**）
- **受託者**: 信託銀行（三井住友信託、みずほ信託、三菱UFJ信託等）に限定
- **受益者**: 特定障害者本人
- **信託できる財産**: 金銭、有価証券、金銭債権、不動産等
- **最低信託金額**: 多くの信託銀行で **1,000万円から**
- **信託設定手数料**: 信託元本の **3.3%（税込）**
- **使途制限**: 生活費・医療費に限定（奢侈品・株式購入等は不可）
- **終了条件**: 受益者（障害者本人）の死亡時に終了
- **途中解約**: 原則不可
- **手続き**: 「障害者非課税信託申告書」を信託銀行経由で所轄税務署に提出

### 4-5. 少額貯蓄非課税制度（マル優・特別マル優）

| 制度 | 対象 | 非課税限度額 |
|------|------|------------|
| マル優（少額貯蓄非課税） | 障害者手帳所持者（等級不問） | 元本 **350万円** までの利子が非課税 |
| 特別マル優（少額公債非課税） | 障害者手帳所持者（等級不問） | 額面 **350万円** までの利子が非課税 |
| **合計** | | **700万円** 分の利子が非課税 |

## 6. 企業（雇用する側）のメリット

### 6-1. 法定雇用率制度（2026年7月〜）

| 項目 | 内容 |
|------|------|
| 法定雇用率 | **2.7%**（2026年7月〜） |
| 対象企業 | 常用労働者 **37.5人以上** |
| 未達成の場合（納付金） | 不足1人あたり **月額5万円**（常用労働者100人超の企業） |
| 改善命令に従わない場合 | **企業名の公表** |

### 6-2. 雇用カウント方法（等級別）

| 障害者区分 | 週30時間以上 | 週20〜30時間未満 | 週10〜20時間未満 |
|-----------|------------|-----------------|-----------------|
| **重度身体障害者**（1〜2級） | **2カウント**（ダブルカウント） | **1カウント** | **0.5カウント** |
| 身体障害者（3〜6級） | 1カウント | 0.5カウント | − |
| **重度知的障害者**（A判定） | **2カウント**（ダブルカウント） | **1カウント** | **0.5カウント** |
| 知的障害者（B判定） | 1カウント | 0.5カウント | − |
| 精神障害者（全等級） | 1カウント | 1カウント（当面の措置） | 0.5カウント |

**ポイント**: 重度障害者（身体1〜2級、療育A）は **ダブルカウント** されるため、法定雇用率達成に大きく貢献する。

### 6-3. 障害者雇用調整金・報奨金

| 制度 | 対象企業 | 金額 |
|------|---------|------|
| **障害者雇用調整金** | 常用労働者100人超 | 法定雇用率超過1人あたり **月額29,000円**（10人超の分は23,000円） |
| **障害者雇用報奨金** | 常用労働者100人以下 | 超過1人あたり **月額21,000円**（35人超の分は16,000円） |

### 6-4. 助成金一覧

#### 特定求職者雇用開発助成金（特定就職困難者コース）

| 対象 | 中小企業 | 大企業 | 助成期間 |
|------|---------|-------|---------|
| 重度障害者等（身体1〜2級 / 知的A） | **240万円** | 100万円 | 3年 |
| 重度障害者等（短時間） | **80万円** | 30万円 | 2年 |
| 重度以外の障害者 | **120万円** | 50万円 | 2年 |
| 重度以外の障害者（短時間） | **80万円** | 30万円 | 2年 |

#### その他の助成金

| 助成金名 | 金額 | 条件 |
|---------|------|------|
| **障害者初回雇用コース（ファースト・ステップ奨励金）** | **120万円** | 初めて障害者を雇用し、法定雇用率を達成した中小企業 |
| **トライアル雇用助成金（障害者トライアルコース）** | 月額最大 **4万円** × 最長6ヶ月 | ハローワーク等の紹介により試行雇用 |
| **トライアル雇用助成金（障害者短時間トライアルコース）** | 月額最大 **4万円** × 最長12ヶ月 | 週10〜20時間からの雇入れ |

| **障害者介助等助成金** | 費用の3/4（上限あり） | 職場介助者の配置等 |
| **職場適応援助者（ジョブコーチ）助成金** | 支援1日あたり定額 | ジョブコーチによる支援を実施 |
| **障害者作業施設設置等助成金** | 費用の2/3（上限あり） | 作業施設等の設置・整備 |

### 6-5. 税制優遇（企業側）

| 項目 | 内容 |
|------|------|
| 機械等の割増償却 | 障害者が常時雇用する労働者数に占める割合が50%以上の場合、機械装置の **24%割増償却** |
| 不動産取得税の減額 | 助成金を受けて事業所を取得した場合の減額措置あり |

### 6-6. 社会的メリット

| メリット | 説明 |
|---------|------|
| CSR・ESGの向上 | 障害者雇用は企業の社会的責任として評価される |
| 入札加点 | 官公庁の入札において障害者雇用実績が加点要素になる場合がある |
| 多様性の推進 | 業務プロセスの見直し・効率化のきっかけになる |
| ブランド価値 | 障害者雇用に積極的な企業としてのイメージ向上 |

### 6-7. 注意事項

| 項目 | 内容 |
|------|------|
| **業務委託はカウント対象外** | コンサル契約（業務委託）では法定雇用率にカウントされない |
| **雇用契約が必要** | 週20時間以上の雇用契約が必要（10時間以上で特定短時間労働者としてカウント可能） |
| **合理的配慮義務** | 企業は障害者に対して合理的配慮を提供する義務がある |
| **解雇制限** | 障害者の解雇には正当な理由が必要で、ハローワークへの届出が必要 |

## 7. 竹尾さんのケース（身体障害者一級一種）

### 該当する区分
- **手帳種別**: 身体障害者手帳
- **等級**: 1級（最重度）→ **特別障害者**
- **種別**: 第1種 → 介護者含む交通割引適用

### 活用できる税制優遇まとめ

| カテゴリ | 制度 | メリット |
|---------|------|---------|
| 贈与税 | 特定贈与信託 | **6,000万円まで非課税**（知人からの贈与もOK） |
| 所得税 | 特別障害者控除 | **40万円の所得控除** |
| 所得税 | マル優 + 特別マル優 | **合計700万円の利子非課税** |
| 住民税 | 特別障害者控除 | **30万円の所得控除** |

| 自動車 | 自動車税減免 | **全額免除**（自治体が多い） |
| 自動車 | 高速道路ETC | **50%割引** |
| 交通 | JR・私鉄・バス | 本人 **50%割引** + **介護者も50%割引**（第1種） |
| 通信 | 携帯電話 | 各キャリアの障害者割引 |
| 放送 | NHK受信料 | 半額免除（世帯全員非課税なら全額免除） |

### 企業側（倉田側）のメリット

| 項目 | メリット |
|------|---------|
| 雇用カウント | **ダブルカウント（2人分）**（週30時間以上の場合） |
| 調整金 | 月額 **29,000円** |
| 助成金 | 最大 **240万円**（中小企業・特定求職者雇用開発助成金） |
| トライアル雇用 | 月額最大 **4万円** × 最長6ヶ月 |
